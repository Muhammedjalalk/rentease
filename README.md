# 🏠 RentEase - Furniture & Appliance Rental Platform

RentEase is a full-stack web application that allows users to rent furniture and home appliances easily.  
It connects users with rental services and provides a smooth booking and management system.

---

## 🚀 Tech Stack

### Frontend:
- React.js
- HTML5
- CSS3
- JavaScript

### Backend:
- Node.js
- Express.js

### Database:
- MongoDB

---

## 📌 Features

### 👤 User Features:
- User registration and login
- Browse furniture & appliances
- View product details
- Add items to cart
- Place rental orders
- Track order status

### 🛠 Admin Features:
- Admin dashboard
- Manage products
- Manage orders
- Add / update / delete items
- View user data
